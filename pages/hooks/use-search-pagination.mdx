# useSearchPagination

`useSearchPagination` provides a small, reusable state model for search + pagination list experiences.

## What It Handles

- current `PaginationParams` state
- query updates through `handleSearch`
- page reset to `1` on every new search
- full param replacement through `handlePaginationChange`

## Usage

```tsx
import { useSearchPagination } from '@rapidset/rapidkit';

export function UsersListState() {
  const { paginationParams, handleSearch, handlePaginationChange } =
    useSearchPagination({ page: 1, size: 20 });

  return (
    <>
      <input
        value={paginationParams.query ?? ''}
        onChange={(event) => {
          handleSearch(event.target.value);
        }}
      />

      <button
        type="button"
        onClick={() => {
          handlePaginationChange({
            ...paginationParams,
            page: (paginationParams.page ?? 1) + 1,
          });
        }}
      >
        Next
      </button>
    </>
  );
}
```
